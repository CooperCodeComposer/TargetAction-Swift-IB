# TargetAction-Swift-IB
App demo of an animation from an array of images 

- Swift 2.2
- Xcode 7.3

##Features:
+ Hours of fun animating through an array of images of the word "Swift"
+ UITextField that changes a UILabel 
+ Layout in interface builder
